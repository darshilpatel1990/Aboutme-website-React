## Description

This project was bootstrapped with Create React App, using the React JavaScript.
It is based on SPA (Single Page Application) model.

React Router enables navigation between views from different components of this application.

### React version of original About me website

Original version of this website was created using Bootstrap only. 
Then, this website is created from scratch by using React with some minor changes in features.

### Features

Initially, there are four page components: `Home`, `Skills`, `Funfacts` and `Contact`.

Languages and features used are as below: 
HTML, CSS, JavaScript, React Components, Reactstrap, useState, Formik, Navbar, Cards etc.
